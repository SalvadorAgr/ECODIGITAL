# Triaje de Issues — Ecodigital

Cuando recibimos tu issue, primero lo triamos. El triaje de un issue generalmente toma alrededor de un día hábil pero puede tomar más tiempo. El objetivo del triaje es proporcionarte una comprensión clara de qué sucederá con tu issue. Por ejemplo, después de que tu solicitud de función fue triada, sabes si planeamos abordar el issue o si escucharemos qué piensa la comunidad más amplia sobre esta solicitud.

Aquí están los estados de los issues y sus descripciones:

| Estado               | Descripción                                                                                                                                                                                                                           |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Untriaged           | El equipo aún no ha revisado el issue. Generalmente lo hacemos dentro de un día hábil.                                                                                                                                                    |
| As designed         | El comportamiento descrito en el issue es intencional. Si encuentras que es seriamente disruptivo o si hemos malinterpretado tu punto, por favor háznoslo saber en la sección de comentarios del issue.                                                            |
| Blocked             | No podemos trabajar en este issue hasta que otro (vinculado) sea resuelto.                                                                                                                                                                   |
| Can't Reproduce     | No hemos podido reproducir el issue de nuestro lado. Podría ser inestable o ya estar corregido, o puede que no hayamos tenido todos los detalles necesarios. Si aún experimentas el issue y tienes más detalles, compártelos. |
| Duplicate           | El issue es el mismo (o tiene la misma causa) que otro (vinculado).                                                                                                                                                                |
| Fixed               | Si el issue era un bug, ha sido corregido; si era una función faltante, ha sido implementada.                                                                                                                                          |
| Fixed In Branch     | Si el issue era un bug, ha sido corregido; si era una función faltante, ha sido implementada; los cambios ahora están en una rama separada y aún no se han fusionado en la rama predeterminada.                                            |
| In Progress         | Estamos trabajando actualmente en el issue.                                                                                                                                                                                                 |
| Incomplete          | Desafortunadamente no tenemos suficiente información para proceder. Si estás dispuesto a compartir más detalles sobre el issue, por favor hazlo en los comentarios.                                                                              |
| Obsolete            | La parte del producto que causaba este issue ha sido eliminada o reestructurada significativamente desde su creación.                                                                                                                  |
| Upvoting            | Actualmente estamos evaluando la demanda del issue y verificando si requiere cambios complicados o riesgosos. Por favor deja un voto o comentario si crees que debería priorizarse.                                                 |
| Open                | Queremos implementar la corrección o función en el futuro cercano. No podemos prometer que aparecerá en el próximo release público, pero está en nuestra lista corta.                                                                                   |
| Shelved             | Hemos revisado el issue y decidido que, aunque tiene mérito, actualmente no podemos incluirlo en nuestro plan a corto plazo.                                                                                                          |
| Third Party Problem | El issue es causado por un tercero. Hemos hecho nuestro mejor esfuerzo para informarles al respecto.                                                                                                                                                    |
| To be Discussed     | Necesitamos tiempo para discutir el issue.                                                                                                                                                                                               |
| To Reproduce        | Intentaremos encontrar los pasos necesarios para reproducir el issue de nuestro lado.                                                                                                                                                              |
| Under Investigation | Hemos triado el issue, pero ahora necesitamos investigarlo más a fondo. Esto puede requerir procesar información adicional como logs o dumps.                                                                                    |
| Waiting for Info    | Hemos solicitado información adicional a la persona que creó el issue y estamos esperando que nos responda.                                                                                                              |
| Declined            | Hemos revisado la sugerencia y, aunque apreciamos su valor, desafortunadamente no tenemos los recursos para implementarla.                                                                                                         |
| Answered            | El issue resultó ser una pregunta o un malentendido, y ha sido respondido o resuelto.                                                                                                                                                           |
