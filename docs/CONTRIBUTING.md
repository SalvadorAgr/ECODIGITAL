# Contribuciones — Ecodigital

¡Las contribuciones son bienvenidas!

Ecodigital es un proyecto desarrollado por **dogma.black**. Si quieres contribuir, sigue estos pasos:

1. Haz un fork del repositorio
2. Crea una rama: `git checkout -b feature/mi-mejora`
3. Realiza tus cambios y haz commit: `git commit -m 'Agrega mi mejora'`
4. Envía tu rama: `git push origin feature/mi-mejora`
5. Abre un Pull Request describiendo los cambios realizados

Para reportes de errores y solicitudes de funciones, abre un [issue](../../issues).
