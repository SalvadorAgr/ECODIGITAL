# Contribuciones

¡Las contribuciones son bienvenidas! Por favor sigue el proceso estándar de GitHub:

1. Haz un fork del repositorio
2. Crea una rama para tu funcionalidad o corrección: `git checkout -b feature/mi-mejora`
3. Realiza tus cambios y haz commit: `git commit -m 'Agrega mi mejora'`
4. Envía tu rama: `git push origin feature/mi-mejora`
5. Abre un Pull Request describiendo los cambios realizados

Para reportes de errores y solicitudes de funciones, abre un [issue](../../issues) en este repositorio.
